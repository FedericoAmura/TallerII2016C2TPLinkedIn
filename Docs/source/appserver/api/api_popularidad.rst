Popularidad
===========

*********************************************
Consulta sobre recomendaciones entre usuarios
*********************************************

+---------------+--------------------------------------------------------------+
|      URL      | http://<server>/users/popular/<userID1>/<userID2>            |
+---------------+--------------------------------------------------------------+
| AUTENTICACIÓN | No                                                           |
+---------------+--------------------------------------------------------------+
|     VERBO     | GET                                                          |
+---------------+--------------------------------------------------------------+
|   PARÁMETROS  |                                                              |

|   RESPUESTA   | - HTTP 200                                                   |
|               | - HTTP 404                                                   |
